# JSON Map View

在浏览器中加载本地 **JSON 地图**（HD Map）、**Layer 导出 JSON**（`*layer_data.json`）与 **TUM 轨迹**，以 3D 视口、场景树与属性面板进行查看与筛选的纯前端工具。数据不离开本机。
